إعدادات الأداء والسرعة

تأثيرات الأنيميشن

GSAP يوفر تأثيرات سلسة ومتطورة. تعطيله يستبدلها بـ CSS فقط ويُحسّن الأداء على الأجهزة الضعيفة.

كاش الـ API

إضافة ترويسات Cache-Control: max-age=300 تخلّي المتصفح والـ CDN يحتفظون بالبيانات 5 دقائق.

ضغط HTML

إزالة المسافات الزائدة من الصفحات الديناميكية مثل category.php يُقلّل حجم الاستجابة.

PHP OPcache

يُسرّع تنفيذ ملفات PHP بتخزينها مُجمَّعة مسبقاً. يُمسح عند الضغط على "مسح الكاش" في أسفل الشريط الجانبي.
